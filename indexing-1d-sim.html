Step 2 — positive indexing: marks[i]
# marks[i] — count from 0, left to right marks[0] → 45 # first element marks[3] → 92 # element at position 3

marks[3] = 92
Position 3 — fourth element.
Step 3 — negative indexing: marks[-i]
# marks[-i] — count from -1, right to left marks[-1] → 72 # last element = marks[n-1] marks[-10] → 45 # = marks[0], wraps to front
 
marks[-1] = 72
Last element — always marks[n-1].
marks[-10] = 45
-n wraps to position 0 — same as marks[0].

# 10 elements: valid range [0..9] or [-10..-1] marks[10] → IndexError: index 10 is out of bounds marks[-11] → IndexError: index -11 is out of bounds # rule: -n ≤ i ≤ -1 OR 0 ≤ i ≤ n-1
Valid: marks[0..9] or marks[-10..-1]
Any index outside this range throws IndexError.
