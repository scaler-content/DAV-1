Part 1 — what is ndim?
# 1D — one row of marks marks = np.array([78, 45, 91]) marks.ndim → 1
→ One straight line of values. ndim = 1 because data flows in one direction only.
# 2D — rows × columns (like a table) results = np.array([[78,92,88],[45,67,34],[91,56,72]]) results.ndim → 2
→ A grid of values. ndim = 2 because data flows in two directions: rows & columns.
