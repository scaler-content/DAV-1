Slicing — grab a RANGE of marks

Speed:
Edit the array below, then press Play.
Step 1 — build your marks array
marks = np.array([45, 78, 23, 92, 67, 88, 34, 56, 91, 72]) # syntax: marks[start : stop : step]
Step 2 — marks[:stop] and marks[start:] — omit one end
# omit start → defaults to 0 | omit stop → defaults to end marks[:3] → [45, 78, 23] # first 3 elements marks[7:] → [56, 91, 72] # from index 7 to end
 
marks[:3] = [45, 78, 23]
First 3 marks — start defaults to 0.
marks[7:] = [56, 91, 72]
Last 3 marks — stop defaults to end.
Step 3 — marks[start:stop] — middle range, stop is EXCLUDED
# marks[2:6] — indices 2, 3, 4, 5 (stop 6 excluded) marks[2:6] → [23, 92, 67, 88] # 4 elements # length = stop − start = 6 − 2 = 4
 
marks[2:6] = [23, 92, 67, 88]
Indices 2 to 5 — 4 elements.
Stop 6 is EXCLUDED — length = 6 − 2 = 4
The slice ends just before stop. Always: stop − start = count.
Step 4 — marks[::step] — pick every step-th element
# marks[start:stop:step] — start/stop default to full range marks[::2] → [45, 23, 67, 34, 91] # every 2nd element marks[1::2] → [78, 92, 88, 56, 72] # alternate group
 
marks[::2] = [45, 23, 67, 34, 91]
Every 2nd element — 5 picks total.
marks[1::2] = [78, 92, 88, 56, 72]
Shift start to 1 — picks the alternate group.
